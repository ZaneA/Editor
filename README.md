Editor
===

An incredibly dumb but self-hosting text editor using [CodeMirror](http://codemirror.net) and [Sinatra](http://sinatrarb.com).

Usage
---

- Clone repository
- Run `bundle install` to get dependencies
- Run `./editor`
- Open `http://127.0.0.1:4567` in your browser
